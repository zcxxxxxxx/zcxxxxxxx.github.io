个人网页

网站建设​

当你想建立个自己的网站玩玩，你可以不必写令人头疼的.html文件，也不需要搞什么乱七八糟的域名，网址啥的。

网页内容​

Docusaurus​

Docusaurus能够帮助你快速创建并发布美观的文档网站。 Docusaurus 是一个静态网站生成器。它将你的网站构建成一个单页面应用程序（single-page application），具有快速地客户端导航功能并充分利用了React的强大能力，为你的网站赋予更好地交互性。虽然 Docusaurus 是为文档功能而生的，但是也可以用来构建任何类型的网站（个人站点、产品介绍、博客、营销页面等）。(个人推选)

Gatsby​

Gatsby自带了许多功能，还拥有丰富的插件生态，能够完成 Docusaurus 所有的功能。当然，这是以更高的学习门槛为代价的。

Next.js​

Next.js是另一个非常流行的混合型 React 框架。它可以辅助你搭建一个好的文档网站，但对于具体到文档这一使用场景来说，要想实现匹配 Docusaurus 一样的开箱即用的功能的话，还需要做大量的开发工作。

Nextra 是一个建立在 Next.js 之上的、具有强制规则（opinionated ）的静态网站生成器。目前，它的功能不如 Docusaurus 完善。
